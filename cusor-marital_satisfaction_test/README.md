# 婚姻满意度测试程序

这是一个基于Web的婚姻满意度测试程序，帮助用户评估在六个维度上的婚姻满意度。

## 功能特点

- **48道测试题目**：涵盖婚姻关系的各个方面
- **六个评估维度**：
  - 亲密关系（8题）
  - 忠诚（7题）
  - 家庭责任（6题）
  - 经济管理（7题）
  - 日常生活（6题）
  - 情感联系（14题）
- **智能计分系统**：自动处理正面和负面题目的计分
- **可视化结果展示**：直观的分数条和详细建议
- **响应式设计**：支持桌面和移动设备

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始测试"按钮
3. 依次回答48道题目，每道题选择最符合您想法的选项
4. 完成所有题目后，点击"查看结果"
5. 查看各维度得分和个性化建议

## 计分规则

- 每道题有7个选项：强烈同意(7分)、同意(6分)、稍微同意(5分)、中立(4分)、稍微不同意(3分)、不同意(2分)、强烈不同意(1分)
- 对于正面题目：强烈同意得分最高
- 对于负面题目：计分反向，强烈同意得分最低
- 每个维度的最终分数 = (平均原始得分 - 1) / 6 × 100
- 分数范围：0-100分
